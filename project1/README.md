SM4.py文件内实现了SM4算法，SM4_addv.py文件内对其又进行了优化。
将S盒从列表改为bytes类型（不可变字节序列），利用底层C实现的查找速度优势，减少Python层面的开销。在__init__中一次性计算所有轮密钥并存储，避免每次加密/解密时重复计算（原版本每次加密都重新计算轮密钥）。使用struct模块的pack/unpack替代手动位操作（如(key[4*i] << 24) | ...），直接处理32位整数组，减少Python循环和位运算次数。通过_uint32函数确保所有中间结果限制在32位，避免意外溢出或符号问题。
将算法封装为SM4类，内部状态（如轮密钥）通过实例属性维护，避免全局变量，更符合工程实践。_pad和_unpad方法独立为静态方法，支持更严格的PKCS#7填充验证（检查填充字节是否一致）。
